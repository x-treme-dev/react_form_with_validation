Создать форму на React для добавления нового проекта по разработке. 
Поля: имя проекта, бюджет в рублях, крайняя дата завершения, эл. почта и телефон руководителя. 
Все поля формы должны быть соответствующего типа (email, tel и т.д.). 
Валидироваь форму в процессе ввода данных с помощью Zod. 
При отправке сохранить данные в localStorage.
